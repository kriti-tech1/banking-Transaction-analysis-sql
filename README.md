# banking-Transaction-analysis-sql
                                          **Project Overview**

.This project analyzes customer banking transactions using PostgreSQL.
.It includes database creation, sample datasets, and SQL queries solving real-world banking business problems.

**Database Tables**
.Customers
.Accounts
.Transactions

**SQL Concepts Used**
.JOINs
.GROUP BY
.HAVING
.CTEs
.Subqueries
.Correlated Subqueries
.Window Functions
.RANK()
.DENSE_RANK()
.LAG()
.LEAD()
.Running Totals
.CASE Statements
.EXISTS / NOT EXISTS

**Business Problems Solved**
.Customers with highest transactions
.Running transaction totals
.Latest transaction tracking
.Deposit vs Withdrawal analysis
.Ranking customers by balance
.Monthly transaction analysis

**Tools Used**
.PostgreSQL
.pgAdmin
