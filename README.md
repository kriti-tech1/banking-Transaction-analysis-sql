# banking-Transaction-analysis-sql
# Banking Transaction Analysis using PostgreSQL

                                                 ## Project Overview ##
This project analyzes customer banking transactions using PostgreSQL. It includes database creation, sample datasets, and SQL queries solving real-world banking business problems.

## Database Tables
- Customers
- Accounts
- Transactions

## SQL Concepts Used
- JOINS
- GROUP BY
- HAVING
- CTEs
- Subqueries
- Correlated Subqueries
- Window Functions
- RANK()
- DENSE_RANK()
- LAG()
- LEAD()
- CASE Statements
- EXISTS / NOT EXISTS

## Business Problems Solved
- Customers with highest transactions
- Running transaction totals
- Latest transaction tracking
- Deposit vs Withdrawal analysis
- Ranking customers by balance
- Monthly transaction analysis

## Tools Used
- PostgreSQL
- pgAdmin

## Project Outcome
Improved SQL problem-solving skills by working on real-world banking transaction scenarios and advanced SQL queries.
